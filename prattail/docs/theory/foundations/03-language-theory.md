# Formal Language Theory for PraTTaIL

This document establishes the formal language theory underpinning PraTTaIL's
architecture: closure properties of context-free languages, the Bar-Hillel
intersection theorem, and term rewriting systems. Each theorem is stated
precisely, proved without gaps, and grounded in the concrete PraTTaIL
implementation.

---

## 1. CFL Closure Properties

Context-free languages (CFLs) are closed under some operations but not others.
The following table summarizes the closure properties relevant to PraTTaIL's
grammar composition pipeline.

| Operation                 | Regular | CFL | Both closed? |
|---------------------------|---------|-----|--------------|
| Union                     | ✓       | ✓   | ✓            |
| Concatenation             | ✓       | ✓   | ✓            |
| Kleene star               | ✓       | ✓   | ✓            |
| Intersection              | ✓       | ✗   | ---          |
| Complement                | ✓       | ✗   | ---          |
| Intersection with regular | ✓       | ✓   | ✓            |
| Homomorphism              | ✓       | ✓   | ✓            |
| Inverse homomorphism      | ✓       | ✓   | ✓            |

**Definition 1.1 (Context-Free Grammar).** A context-free grammar is a
4-tuple G = (V, Σ, P, S) where V is a finite set of nonterminals, Σ is a
finite terminal alphabet with V ∩ Σ = ∅, P ⊆ V × (V ∪ Σ)* is a finite set
of productions, and S ∈ V is the start symbol. The language L(G) is the set
of all strings w ∈ Σ* such that S ⇒* w.

**Theorem 1.2 (CFL Closure under Union).** If L₁ and L₂ are context-free
languages, then L₁ ∪ L₂ is context-free.

**Proof.** Let G₁ = (V₁, Σ, P₁, S₁) and G₂ = (V₂, Σ, P₂, S₂) be CFGs for
L₁ and L₂ respectively, with V₁ ∩ V₂ = ∅ (rename if necessary). Construct
G' = (V₁ ∪ V₂ ∪ {S'}, Σ, P₁ ∪ P₂ ∪ {S' → S₁, S' → S₂}, S') where
S' ∉ V₁ ∪ V₂. For any w ∈ Σ*: w ∈ L(G') ⟺ S' ⇒ S₁ ⇒* w or
S' ⇒ S₂ ⇒* w ⟺ w ∈ L₁ ∪ L₂. Since G' is a CFG, L₁ ∪ L₂ is
context-free. QED

**Theorem 1.3 (CFL Non-Closure under Intersection).** The class of
context-free languages is not closed under intersection.

**Proof.** Consider L₁ = {aⁿbⁿcᵐ | n, m ≥ 0} and L₂ = {aᵐbⁿcⁿ | n, m ≥ 0}.
Both are context-free: L₁ is generated by S → AB, A → aAb | ε, B → cB | ε;
and L₂ is generated by S → AB, A → aA | ε, B → bBc | ε. However,
L₁ ∩ L₂ = {aⁿbⁿcⁿ | n ≥ 0}, which is not context-free (by the pumping
lemma for CFLs: pumping any segment of aⁿbⁿcⁿ for sufficiently large n
disrupts the equality of exponents). Since two CFLs intersect to produce a
non-CFL, the class is not closed under intersection. QED

**Concrete Grounding.** PraTTaIL's `extends` and `compose` operations (in the
macros crate) use exclusively CFL-safe operations:

- **`extends`**: Takes a base language and adds new production rules. This is
  union of production sets: the resulting grammar generates L_base ∪ L_extension,
  which is CFL by Theorem 1.2.
- **`includes`**: Imports types and terms from another language specification,
  which amounts to union of nonterminal definitions --- again CFL-safe.
- **Cross-category dispatch via cast rules**: When the parser switches between
  categories (e.g., from Expr to Name), it selects among grammar alternatives.
  This is grammar union at the rule level, not intersection.
- **No intersection or complement**: None of PraTTaIL's composition mechanisms
  require CFL ∩ CFL or CFL complement, so the well-formedness of the composed
  grammar is preserved by the closure properties above.

## 2. Bar-Hillel Theorem (CFL ∩ Regular = CFL)

**Theorem 2.1 (Bar-Hillel, Perles, Shamir 1961).** If L₁ is a context-free
language and L₂ is a regular language, then L₁ ∩ L₂ is context-free.

**Proof (Product Construction).** Let G = (V, Σ, P, S) be a CFG generating L₁
and M = (Q, Σ, δ, q₀, F) be a DFA recognizing L₂.

*Construction.* Define a new CFG G' = (V', Σ, P', S') as follows:

- **Nonterminals:** V' = {[A, p, q] | A ∈ V, p ∈ Q, q ∈ Q}. Each triple
  [A, p, q] represents "nonterminal A derives a string that takes DFA state p
  to state q."

- **Terminal productions:** For each production A → a where a ∈ Σ, and for
  each pair of states p, q ∈ Q with δ(p, a) = q, add the production
  [A, p, q] → a to P'.

- **Nonterminal productions:** For each production A → B₁B₂...Bₖ where each
  Bᵢ ∈ V ∪ Σ, and for all state sequences p₀, p₁, ..., pₖ ∈ Q, add the
  production [A, p₀, pₖ] → [B₁, p₀, p₁][B₂, p₁, p₂]...[Bₖ, pₖ₋₁, pₖ]
  to P', where terminal symbols Bᵢ = a are handled by requiring
  δ(pᵢ₋₁, a) = pᵢ and emitting a directly.

- **Start symbols:** S' is a fresh nonterminal with productions
  S' → [S, q₀, qf] for each qf ∈ F.

*Correctness.* We prove L(G') = L₁ ∩ L₂ by showing both inclusions.

(⊆) Suppose w ∈ L(G'). Then S' ⇒ [S, q₀, qf] ⇒* w for some qf ∈ F. By
induction on the derivation, [A, p, q] ⇒* u implies A ⇒* u in G (stripping
the state annotations) and δ*(p, u) = q (each terminal step respects δ).
Therefore S ⇒* w (so w ∈ L₁) and δ*(q₀, w) = qf ∈ F (so w ∈ L₂).

(⊇) Suppose w ∈ L₁ ∩ L₂. Then S ⇒* w in G with some parse tree T, and
δ*(q₀, w) = qf ∈ F. Annotate each node A in T with the DFA state interval
[p, q] determined by the yield of the subtree rooted at A: p is the DFA state
before reading the yield, q is the state after. This annotation is well-defined
because M is deterministic. The annotated tree is a valid derivation in G'
from [S, q₀, qf], so w ∈ L(G').

*G' is a CFG:* Every production in P' has a nonterminal on the left and a
string of nonterminals and terminals on the right, satisfying the CFG form.
Therefore L₁ ∩ L₂ is context-free. QED

**Concrete Grounding.** This theorem justifies PraTTaIL's two-phase
lexer-then-parser pipeline architecture:

- The **DFA lexer** (constructed in `automata/subset.rs` via subset
  construction from Thompson NFAs) recognizes a regular language: the set of
  all valid token streams over the input alphabet.
- The **Pratt + recursive-descent parser** (implemented in `trampoline.rs`)
  accepts a context-free language defined by the grammar specification.
- Their interaction --- parsing the token stream produced by the lexer --- is
  semantically an intersection of a CFL with a regular language. By
  Theorem 2.1, the result is context-free.
- This guarantees that the two-phase decomposition loses no expressiveness
  compared to a monolithic CFL parser: every language expressible as CFL ∩ REG
  is still a CFL.
- In practice, the DFA lexer feeds tokens to the trampoline parser, and the
  WFST dispatch layer (in `wfst.rs`) assigns weights to token alternatives.
  The weight selection is a refinement within the regular token language, so
  the intersection remains CFL.

## 3. Rewriting Systems

### 3.1 Definitions

**Definition 3.1 (Signature).** A signature Σ is a finite set of function
symbols, each with an associated arity: Σ = {f₁/n₁, f₂/n₂, ..., fₖ/nₖ}
where nᵢ ∈ ℕ is the arity of fᵢ. Symbols of arity 0 are constants.

**Definition 3.2 (Term).** Given a signature Σ and a countably infinite set of
variables X, the set of terms T(Σ, X) is the smallest set such that:
- x ∈ T(Σ, X) for every x ∈ X
- f(t₁, ..., tₙ) ∈ T(Σ, X) for every f/n ∈ Σ and t₁, ..., tₙ ∈ T(Σ, X)

**Definition 3.3 (Substitution).** A substitution σ is a mapping from
variables to terms, σ: X → T(Σ, X), extended homomorphically to all terms:
σ(f(t₁, ..., tₙ)) = f(σ(t₁), ..., σ(tₙ)).

**Definition 3.4 (Term Rewriting System).** A term rewriting system (TRS) is a
pair (Σ, R) where Σ is a signature and R is a finite set of rewrite rules
l → r such that:
- l, r ∈ T(Σ, X) (both sides are terms over Σ and variables)
- l ∉ X (the left-hand side is not a bare variable)
- Var(r) ⊆ Var(l) (every variable in r appears in l)

A term s rewrites to t in one step (written s →_R t) if there exists a
position p in s, a rule l → r ∈ R, and a substitution σ such that
s|_p = σ(l) and t = s[p ← σ(r)]. We write →*_R for the reflexive-transitive
closure.

### 3.2 Termination

**Definition 3.5 (Strong Normalization).** A TRS (Σ, R) is strongly
normalizing (terminating) if there exists no infinite rewrite chain
t₁ →_R t₂ →_R t₃ →_R ...

**Definition 3.6 (Term Size).** The size of a term t, written |t|, is defined
inductively: |x| = 1 for variables x ∈ X, and |f(t₁, ..., tₙ)| = 1 + Σᵢ|tᵢ|
for compound terms.

### 3.3 Confluence

**Definition 3.7 (Confluence / Church-Rosser Property).** A TRS (Σ, R) is
confluent if for all terms s, t₁, t₂ ∈ T(Σ, X):
  s →*_R t₁ ∧ s →*_R t₂ ⟹ ∃ u ∈ T(Σ, X) . t₁ →*_R u ∧ t₂ →*_R u

**Definition 3.8 (Local Confluence).** A TRS is locally confluent if for all
terms s, t₁, t₂: s →_R t₁ ∧ s →_R t₂ ⟹ ∃ u . t₁ →*_R u ∧ t₂ →*_R u

**Theorem 3.9 (Newman's Lemma, 1942).** A terminating TRS is confluent if
and only if it is locally confluent.

**Proof.** The forward direction is immediate (confluence implies local
confluence by restricting to single steps). For the reverse, assume the TRS
is terminating and locally confluent but not confluent. Then there exist terms
s, t₁, t₂ with s →*_R t₁, s →*_R t₂, and no common reduct. Among all such
counterexamples, choose s to be minimal with respect to the termination
ordering (which is well-founded by the termination hypothesis). Since s →*_R t₁
and s →*_R t₂, we can write s →_R s₁ →*_R t₁ and s →_R s₂ →*_R t₂ (or one
of the paths is zero-length, in which case confluence is trivially satisfied).
By local confluence, s₁ and s₂ have a common reduct v: s₁ →*_R v and
s₂ →*_R v. Now s₁ is strictly smaller than s in the termination ordering, and
s₁ →*_R t₁ and s₁ →*_R v, so by minimality of s, t₁ and v have a common
reduct u₁. Similarly, s₂ →*_R t₂ and s₂ →*_R v, so t₂ and v have a common
reduct u₂. Finally, u₁ and u₂ are both reducts of v, and v is strictly smaller
than s, so they have a common reduct u. Then t₁ →*_R u₁ →*_R u and
t₂ →*_R u₂ →*_R u, contradicting our assumption that no common reduct exists.
QED

**Concrete Grounding.** Newman's Lemma is the theoretical foundation for
PraTTaIL's `normalize` function correctness. The normalize function applies
rewrite rules eagerly until a normal form is reached. Newman's Lemma guarantees
that if the rewrite system terminates and is locally confluent, the normal form
is unique regardless of the order rules are applied --- which means PraTTaIL's
generated normalize function is deterministic.

### 3.4 Cancellation Pairs

**Definition 3.10 (Cancellation Pair).** Given constructors Outer/1 and
Inner/1 in a signature Σ, a cancellation pair is a rewrite rule of the form
Outer(Inner(x)) → x where x ∈ X is a variable. The pair (Outer, Inner) forms
an involution: composing both constructors cancels to the identity on x.

**Theorem 3.11 (Termination of Cancellation Normalization).** Let (Σ, R) be a
TRS where every rule in R has the form Cᵢ(Dᵢ(x)) → x for some constructors
Cᵢ, Dᵢ ∈ Σ. Then (Σ, R) is strongly normalizing.

**Proof.** Define the measure μ: T(Σ, X) → ℕ as the term size |t| (Definition
3.6). Consider an arbitrary rewrite step t →_R t' at position p using rule
Cᵢ(Dᵢ(x)) → x with substitution σ. The redex at position p is
Cᵢ(Dᵢ(σ(x))), which has size |Cᵢ(Dᵢ(σ(x)))| = 1 + 1 + |σ(x)| = 2 + |σ(x)|.
It is replaced by σ(x), which has size |σ(x)|. Therefore
|t'| = |t| - 2 < |t|, so μ strictly decreases. Since μ maps to ℕ, which is
well-ordered, there can be no infinite descending chain. QED

**Theorem 3.12 (Confluence of Non-Overlapping Cancellation Pairs).** Let
(Σ, R) be a TRS where every rule has the form Cᵢ(Dᵢ(x)) → x, and all outer
constructors Cᵢ are pairwise distinct. Then (Σ, R) is confluent.

**Proof.** By Theorem 3.11, the system terminates. By Newman's Lemma
(Theorem 3.9), it suffices to show local confluence. Consider a term s with
two distinct one-step reducts: s →_R t₁ (rewriting at position p₁) and
s →_R t₂ (rewriting at position p₂). There are two cases:

*Case 1: p₁ and p₂ are non-overlapping positions.* The two rewrites act on
disjoint subterms, so both can be applied independently. Define u by applying
both rewrites: t₁ →_R u and t₂ →_R u.

*Case 2: One position is a prefix of the other.* Without loss of generality,
p₁ is a prefix of p₂. The redex at p₁ is Cᵢ(Dᵢ(s₁)) and the redex at p₂
lies within s₁ (inside the Dᵢ argument). Since the outer constructors Cᵢ are
pairwise distinct, the only rule with outer constructor Cᵢ is the one at p₁.
The redex at p₂ must use a different rule Cⱼ(Dⱼ(...)) → ... with j ≠ i,
so it lies strictly inside the subterm s₁. Rewriting at p₁ produces s₁
(erasing the Cᵢ(Dᵢ(...)) wrapper), and the redex at p₂ still exists within
s₁, so we can rewrite it to obtain u. Symmetrically, rewriting at p₂ first
modifies s₁ to s₁', and the redex Cᵢ(Dᵢ(s₁')) at p₁ still matches (the rule
Cᵢ(Dᵢ(x)) → x matches any argument), producing s₁'. Then s₁' →_R u and
s₁ →_R u by the inner rewrite. Both paths converge to u.

In both cases, a common reduct u exists. Therefore the system is locally
confluent, and by Newman's Lemma, confluent. QED

**Concrete Grounding.** PraTTaIL's `CancellationPair` detection (in
`macros/src/ast/pattern.rs`) scans equations at compile time for the pattern
Outer(Inner(X)) = X. Two concrete instances arise in the RhoCalc language:

- **ExecEq**: `PDrop(NQuote(P)) = P` --- executing a quoted process cancels
  the quote.
- **QuoteDrop**: `NQuote(PDrop(N)) = N` --- quoting a dropped name cancels
  the drop.

Here `PDrop` and `NQuote` are distinct outer constructors, so Theorem 3.12
applies: the cancellation rewrite system is confluent with unique normal forms.

These pairs are **suppressed from the `eqrel` relation** (in `logic/mod.rs`)
because adding Outer(Inner(x)) ≡ x to an equivalence relation triggers
unbounded exploration: the Ascent engine would repeatedly wrap and unwrap terms,
never reaching a fixed point. Instead, they are emitted as `normalize` arms
(in `gen/term_ops/normalize.rs`): the pattern `Outer(Inner(x)) => x` fires
eagerly, maintaining the invariant that no term in the working set contains a
cancellable redex. Theorems 3.11 and 3.12 guarantee that this normalization
terminates and produces unique results.

The lint system reports these detections: **G25** (note) confirms a
cancellation pair was found, and **W09** (warning) alerts when a detected pair
lacks a corresponding rewrite rule in the specification.
