# PD04: missing-feature-gate

**Severity:** Warning
**Module:** Predicate Dispatch
**Feature gate:** `predicate-dispatch`
**Source:** `prattail/src/lint.rs` → `lint_pd04_missing_feature_gate()`

## What It Detects

A predicate guard requires a module whose Cargo feature is not enabled. This
means the dispatch system detected a variety requirement that cannot be fulfilled
at compile time.

Currently PD04 specifically detects cross-channel predicates (requiring M11
Two-Way Transducer) when the `two-way-transducer` feature is not enabled.

## Example

### Triggering Code

```toml
# Cargo.toml
[features]
predicate-dispatch = ["symbolic-automata", "weighted-mso"]
# Note: two-way-transducer is NOT enabled
```

```rholang
for (@x <- ch1, @y : related(x) <- ch2) { P }
```

### Diagnostic Output

```
warning[PD04]: cross-channel predicate detected but 'two-way-transducer' feature
               is not enabled
  = note: predicate references variable 'x' from channel 'ch1' in guard on 'ch2'
  = help: enable the 'two-way-transducer' feature in Cargo.toml to analyze
          bidirectional channel constraints
```

### Fix

Enable the required feature in `Cargo.toml`:

```toml
[dependencies.mettail-prattail]
features = ["predicate-dispatch", "two-way-transducer"]
```

Or use the `full-analysis` convenience feature which includes all modules:

```toml
[dependencies.mettail-prattail]
features = ["full-analysis"]
```

## Rationale

Cross-channel constraints are a core Rholang pattern (join receives with
inter-channel dependencies). Without the Two-Way Transducer module, PraTTaIL
cannot detect deadlock cycles (TW01) or optimize bidirectional constraint
propagation. The warning ensures users are aware of this analysis gap.

## When It Does Not Fire

- When `two-way-transducer` feature is enabled
- When no cross-channel predicate is detected (M11 bit not set in signature)

## Related Diagnostics

- **TW01**: Deadlock cycle detected (only available with `two-way-transducer`)
- **TW02**: One-way equivalence opportunity
- **MT01**: Disconnected tape detected (related multi-tape analysis)
