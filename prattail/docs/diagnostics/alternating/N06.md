# N06: weighted-parity-non-convergent

**Severity:** Warning
**Module:** M3 Weighted Alternating Automata
**Feature gate:** `alternating`

## What It Detects

- The alternating automaton has many non-bisimilar pairs relative to its state count, indicating that the parity game value computation may not converge quickly.
- Specifically, the lint fires when there are more than 3 non-bisimilar pairs and more than 5 states. A high density of structural divergence across categories means the fixpoint iteration for parity game values requires more rounds to stabilize.

```
        PAs   = <ident> "@" Pat;
    }
    // ... 6+ structurally distinct categories with > 3 non-bisimilar pairs

```

### Diagnostic Output

```
warning[N06] (GameLang): alternating automaton has 7 non-bisimilar pairs across 8 states -- parity game value may not converge quickly
  = hint: consider bounding the alternation depth or using a fixpoint iteration limit
```

### Fix

```
// Option 1: Reduce category count by merging structurally similar categories.
// Option 2: Add a fixpoint iteration limit in the analysis configuration:
//   PRATTAIL_PARITY_ITER_LIMIT=100
// Option 3: Simplify branching structure so fewer categories are
//   adversarially different.
```

## Rationale

- In the Kostolányi & Misun formulation, parity game values are computed via semiring-lifted Jurdzinski small progress measures. The convergence rate depends on the number of structurally distinct behaviors: more non-bisimilar pairs means more iterations before the fixpoint stabilizes.
- The polynomial transition functions `psi[p,c] in S[x_1,...,x_n]` require evaluation at each state. When many state pairs are non-bisimilar, the polynomial system has fewer symmetries to exploit, slowing convergence.
- The threshold (`pair_count > 3 && state_count > 5`) avoids noise on small grammars where non-bisimilarity is expected and convergence is fast regardless.
- For non-locally-finite semirings (e.g., `TropicalWeight`), alternating weighted automata are strictly more powerful than non-alternating WA (Theorem 7.1), but this added power comes with slower convergence.
