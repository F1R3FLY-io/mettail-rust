# N07: weighted-branching-imbalance

**Severity:** Note
**Module:** M3 Weighted Alternating Automata
**Feature gate:** `alternating`

## What It Detects

- The alternating automaton is purely adversarial: every pair of states is non-bisimilar. No two categories in the grammar have structurally equivalent observable behavior.
- This means the grammar has maximum structural diversity -- every category is distinguishable from every other, and no merging opportunities exist from a bisimulation perspective.

## Example

### Triggering Code

```
language! {
    name: Diverse;
    primary: Expr;

    category Expr {
        Add = Expr "+" Term;
        Lit = <int>;
    }

    category Term {
        Mul = Term "*" Factor;
        Neg = "-" Factor;
    }

    category Factor {
        Paren = "(" Expr ")";
        Var   = <ident>;
    }
    // All 3 pairs (Expr,Term), (Expr,Factor), (Term,Factor) are non-bisimilar
}
```

### Diagnostic Output

```
note[N07] (Diverse): alternating automaton is purely adversarial (3 states, all pairs non-bisimilar)
  = hint: all categories have structurally different behavior; consider if some can be merged
```

### Fix

```
// If categories are intentionally distinct, no action needed -- the note
// simply documents the structural diversity.

// If some categories should be interchangeable, add rules to make
// their transition structures match:
category Term {
    Mul = Term "*" Factor;
    Neg = "-" Factor;
    Add = Term "+" Factor;  // mirror Expr's addition rule
}
```

## Rationale

- The lint fires when `total_pairs > 0 && non_bisimilar_pairs.len() == total_pairs`, where `total_pairs = n*(n-1)/2` for `n` states. This is the maximum structural divergence.
- In the two-player bisimulation game, the Attacker wins from every initial position pair. This means the Defender has no winning strategy anywhere -- the grammar categories are maximally distinguishable.
- For universal (product) states in the alternating automaton, branching imbalance means successor weights can differ by orders of magnitude. When all branches have structurally different costs, the universal conjunction `times` over successors amplifies the worst-case cost.
- This is a note rather than a warning because maximum diversity is often the natural state of a well-designed grammar where each category serves a distinct syntactic role.
